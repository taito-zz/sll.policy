Log in as the portal owner::

    >>> from plone.app.testing import TEST_USER_NAME, TEST_USER_PASSWORD
    >>> browser.login(TEST_USER_NAME, TEST_USER_PASSWORD)

Go to manage-portlets page and check that 'portlets.Login' is not there::

    >>> browser.open('/@@manage-portlets')
    >>> 'portlets.Login' in browser.contents
    False

Check that there are right portlet column::

    >>> 'id="portletmanager-plone-rightcolumn"' in browser.contents
    True

Go to subfolder and check that there are no right column::

    >>> browser.open('/liity/@@manage-portlets')
    >>> 'id="portletmanager-plone-rightcolumn"' in browser.contents
    False
